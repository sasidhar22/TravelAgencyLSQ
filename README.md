## 📋 About

LSQTravelAgency is a front-end project simulating a travel agency website. It promotes a curated Ladakh travel package with a full itinerary, pricing, testimonials, and an Instagram-style photo gallery.

## 🚀 Features

- **Hero section** with package pricing and CTA
- **Day-by-day itinerary** with images
- **Traveller testimonial** section
- **Instagram photo gallery**
- **Responsive navbar** with navigation links
- **Sticky offer banner** with booking deadline

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
